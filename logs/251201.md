# espidf插件尝试 
1. 安装使用
   1. 下载插件，advanced，config 第三个选项即本地导入
   2. 关机重启
   3. 创建工程，template-app,新建工程
2. 头文件路径，在vscode./c_cpp_properities添加
```bash
"includePath": [
    "${workspaceFolder}/**",
    "${workspaceFolder}/components/**",
    "${workspaceFolder}/main",
    "/home/user/esp/esp-idf/components/**"
]
```
3. 配置芯片时钟节拍
   1. 配置封装外flah：
      1. serial flash configure
      2. flash SPI mode :QIO(Qiud SPI)
      3. flash SPI speed :80mhz 
      4. flash size :16mb
   2. 配置封装内psram
      1. esp psram,勾选(35,36,37引角不能用)
      2. SPI RAM config :Octal
      3. clock speed :80mhz
   3. 配置主频
      1. esp system setting :240mhz
   4. 配置freertos时钟节拍
      1. Kernel :configTIVK_RATE_HZ:1000
4. en和boot电路讲解
   1. en键集成电阻，按下时en引脚接3.3v，系统复位
   2. boot是按下时根据0 3 45 46引脚电平进行选择模式
   3. com下载是一键下载
5. 常用的函数可以自己封装到component引用
6. 函数返回值判断是否成功
   1. ESP_LOGI("main","imformation")打印信息到标签"main"
   2. ESP_LOGE()打印错误
7. 时钟系统，有两个，内部自动选择，外设时钟需要手动选择
8. 定时器
   1. 有两组定时器，分别是两个通用定时器和一个主系统看门狗定时器
   2. 先要配置时钟源
   3. 生成计数器时钟TB_CLK，，用于驱动计数器工作
   4. 计数器的值会实时传送给比较器，值大的时候会发生事件
   5. 当重置计数器和计数器的链接时，会重置，就能发生多次事件，循环计数
9. PWM实现脉冲转换
   1.  MC_PWM根据占空比实现角度速度控制，angle上限180,quint16_t duty=((float)angle/1800+0.0025)*1024

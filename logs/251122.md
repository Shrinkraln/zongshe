5. 出现IDF_PATH未找到可能是CMakeList.tct文件爱里面的环境变量未及时改正，需要传入的是环境变量，而不是普通变量
```bash
cmake_minimum_required(VERSION 3.5)

if(NOT DEFINED ENV{IDF_PATH})
    message(FATAL_ERROR "The CMake varible IDF_PATH was not set!")
endif()

if(NOT DEFINED ENV{UROS_APP})
    message(FATAL_ERROR "The CMake varible UROS_APP was not set!")
endif()

include($ENV{IDF_PATH}/tools/cmake/project.cmake)
project($ENV{UROS_APP})
```
